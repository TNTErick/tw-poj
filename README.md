# tw-poj
MSKLC Layout for freely typing accented Latin scripts, including Pe̍h-ōe-jī and Tâi-lô among others.

# Phah ē-chhut ê jī-hû
í-hā só kóng ê `<ALT>` sī **chiàⁿ-pêng ê Alt**-khí. Nā-sī siūⁿ-beh iōng tò-chhiú ê, ài khà **Ctrl kap Alt** 2-hê.

## Pe̍h-ōe-jī Tiāu-hû
1. Chiàⁿ-pêng ALT ka sò͘-jī (2,3,5,6,7,8,9) chhut tiāu-hû.
2. o͘ e͘ ê tiám ài khà `<ALT>+<KÙ-TIÁM>`; koân ê sè-pún-jī én-nuh (ⁿ) ài khà `<ALT>+n`.
3. Choân-khiu ê nā siá o̤, ṳ (KIP ê er, ir) sī `<ALT>-4`, kâng-khoán tòe tī jī-bó āu-piah.
4. Kah nā Hái-lio̍k ê Kheh Pe̍h-ōe, téng-thâu ê îⁿ-kho͘ (å) iōng `<ALT>+o`.

Phì-lūn kóng nā-sī beh siá "chiâⁿ-súi", ài-khà `c h i a <ALT>+5 <ALT>+n <CHI̍T-UI̍H> s u <ALT+2> i`.
